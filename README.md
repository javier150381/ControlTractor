# TractorPro System 🚜

Sistema profesional para el control de operaciones, ingresos y rentabilidad de maquinaria pesada (tractores).

## 🚀 Características principales
- **Dashboard en Tiempo Real:** Visualización instantánea de ingresos, costos de diésel y ganancia neta.
- **Gestión de Trabajos:** Registro detallado de horas trabajadas por cliente.
- **Control de Diésel:** Seguimiento preciso del consumo y gasto de combustible.
- **Reportes Avanzados:** Corte de caja detallado (Ingresos - Gastos - Entregas - Deudas).
- **Control de Deudores:** Módulo para seguimiento y cobro de cuentas pendientes.
- **Roles de Usuario:** Diferenciación entre Administrador y Operador.

## 🛠️ Tecnologías utilizadas
- **Frontend:** HTML5, Vanilla CSS, JavaScript (ES6+).
- **Backend:** Supabase (Base de datos y RLS).
- **Iconografía:** Lucide Icons.
- **Tipografía:** Google Fonts (Outfit & Inter).

## 📦 Instalación
1. Clona el repositorio.
2. Abre `index.html` en tu navegador (o usa Live Server).
3. Configura las tablas en Supabase usando el archivo `setup.sql`.
